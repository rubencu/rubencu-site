# Icon Files Needed

To complete the SEO setup, you'll need to create the following icon files in the `/public` directory:

## Required Icons

1. **favicon.ico** (16x16 or 32x32) - Main favicon
2. **favicon-16x16.png** (16x16) - PNG favicon for modern browsers
3. **favicon-32x32.png** (32x32) - PNG favicon for modern browsers
4. **apple-touch-icon.png** (180x180) - iOS home screen icon
5. **android-chrome-192x192.png** (192x192) - Android icon
6. **android-chrome-512x512.png** (512x512) - Android icon (large)
7. **og-image.jpg** (1200x630) - Open Graph/social media preview image

## Quick Generation Options

### Option 1: Use a favicon generator
- Visit https://realfavicongenerator.net/
- Upload a logo or image
- Download and extract all files to `/public`

### Option 2: Use a design tool
- Create a simple design in Figma/Canva/Photoshop
- Export at the sizes listed above
- Save to `/public`

### Option 3: Simple placeholder
For now, you can create a simple colored square with your initials "RC" using any image editor.

## Open Graph Image (og-image.jpg)

This is the most important for social sharing. It should be:
- 1200x630 pixels
- Include your name and title
- Look professional (it appears when sharing on LinkedIn, Twitter, etc.)

You can create this using:
- Canva (free templates for social media)
- Figma (design from scratch)
- Any image editor with your photo and text overlay
